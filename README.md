# 🎬 VideoMaster Downloader

Aplicación web para descargar videos y audio (MP3) desde URLs en múltiples formatos y calidades.

## 🚀 Características

- ✅ Soporte para video: MP4, AVI, MOV, MKV
- ✅ Soporte para audio: MP3, M4A, OGG, WAV
- ✅ Calidades de video: 144p hasta 1080p
- ✅ Calidades de audio: 64kbps hasta 320kbps
- ✅ Barra de progreso en tiempo real
- ✅ Historial de descargas (localStorage)
- ✅ Diseño responsive y moderno

## 📦 Instalación

```bash
# Clonar o crear el proyecto
mkdir videomaster-downloader
cd videomaster-downloader

# Instalar dependencias
npm install

# Iniciar servidor
npm start
